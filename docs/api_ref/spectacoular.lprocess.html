lprocess¶

Implements classes for the use in live processing applications. Some of the classes might move to Acoular module in the future.

TimeSamplesPhantom

TimeSamples derived class for propagating signal processing blocks with user-defined time delay.

TimeInOutPresenter

TimeInOut derived class for building an interface from Acoular’s generator pipelines to Bokeh’s ColumnDataSource model that serves as a source for plots and tables.

CalibHelper

Class for calibration of individual source channels

FiltOctaveLive

Octave or third-octave filter (not zero-phase).

TimeSamplesPlayback

Naive class implementation to allow audio playback of .h5 file contents.

SpectraInOut

Provides the spectra of multichannel time data.
