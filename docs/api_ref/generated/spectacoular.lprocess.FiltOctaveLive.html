FiltOctaveLive¶

class spectacoular.lprocess.FiltOctaveLive¶

Bases: acoular.tprocess.FiltFiltOctave, spectacoular.factory.BaseSpectacoular

Octave or third-octave filter (not zero-phase).

This class is similar to Acoular’s FiltFiltOctave. The only difference is that the filter coefficients can be changed while the result function is executed.

result(num)¶

Python generator that yields the output block-wise.

Parameters
numinteger

This parameter defines the size of the blocks to be yielded (i.e. the number of samples per block).

Returns
Samples in blocks of shape (num, numchannels).

Delivers the bandpass filtered output of source. The last block may be shorter than num.
