Reference Manual¶

The following modules are part of SpectAcoular:

bokehview

Implements trait-widget mapping functionality to Acoular classes.

factory

Implements factory for trait to widget translation.

consumer

Implements consumer classes that provide Acoular data to Bokeh data sources

lprocess

Implements classes for the use in live processing applications.

dprocess

Implements data processing classes.

controller

Implements controller classes and functions
