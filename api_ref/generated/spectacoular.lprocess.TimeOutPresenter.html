TimeOutPresenter¶

class spectacoular.lprocess.TimeOutPresenter¶

Bases: TimeOut, BasePresenter

TimeOut derived class for building an interface from Acoular’s generator pipelines to Bokeh’s ColumnDataSource model that serves as a source for plots and tables.

ColumnDataSource is updated from result function. Can be used for automatic presenting of live data.

cdsource = Instance(ColumnDataSource, kw={'data': {'data': np.array([])}})¶

Bokeh’s ColumnDataSource, updated from result loop

result(num)¶

Python generator that yields the output block-wise.

Parameters:
numinteger, defaults to 128

This parameter defines the size of the blocks to be yielded (i.e. the number of samples per block) .

Returns:
Samples in blocks of shape (num, num_channels).

The last block may be shorter than num.
